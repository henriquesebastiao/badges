## Usage üìñ

To search for a specific badge among all those listed below press `Ctrl + F` (or `CMD + F` on MAC OS) and a search box
will open in your browser.

It is possible to use different styles in badges through query parameters of the
powerful [Shields.io](https://shields.io/) API, see the examples below.

### Styles üé®

- `style`: (optional) The style of the badge. One of:
    - `flat` (default)
    - `flat-square`
    - `social`
    - `plastic`
    - `for-the-badge`

Examples:

| Style           | Badge                                                                                                 | URL                                                                                 |
|-----------------|-------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------|
| `flat`          | ![Github](https://img.shields.io/badge/GitHub-100000?logo=github&logoColor=white)                     | `https://img.shields.io/badge/GitHub-100000?style=flat&logo=github&logoColor=white` |
| `flat-square`   | ![Github](https://img.shields.io/badge/GitHub-100000?style=flat-square&logo=github&logoColor=white)   | `https://img.shields.io/badge/GitHub-100000?style=flat&logo=github&logoColor=white` |
| `social`        | ![Github](https://img.shields.io/badge/GitHub-100000?style=social&logo=github&logoColor=black)        | `https://img.shields.io/badge/GitHub-100000?style=flat&logo=github&logoColor=white` |
| `plastic`       | ![Github](https://img.shields.io/badge/GitHub-100000?style=plastic&logo=github&logoColor=white)       | `https://img.shields.io/badge/GitHub-100000?style=flat&logo=github&logoColor=white` |
